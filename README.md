# Ticketing & Visa Data Analysis

## Project Overview
This project focuses on analyzing ticketing and visa operations data for **Dream Line Travel & Tours – Shabiya Branch, Abu Dhabi**.  
The goal is to identify business trends, evaluate salesman performance, and present insights using interactive Power BI dashboards.

---

## Project Objective
- Analyze ticketing and visa datasets  
- Identify key sales trends and patterns  
- Evaluate salesman performance  
- Build interactive dashboards using Power BI  
- Generate actionable business insights  

---

## Data Cleaning & Transformation
Data preprocessing was performed using:
- **Microsoft Excel**
- **Python (Pandas)**
- **Power BI**

### Key Cleaning Steps:
- Removed irrelevant columns (e.g., visa extension details)
- Handled missing values in sales columns
- Cleaned inconsistent values (e.g., `980+1035`, `930+Fine`, `?`)
- Standardized date formats using Pandas
- Filtered unnecessary statuses (e.g., Refund, No Sale, Deleted)
- Combined monthly datasets (Jan & Feb)

## Key Insights

- Visa services contribute the **major share of revenue (~85%)**
- Ticket sales contribute a **smaller portion (~15%)**
- Sales peaked significantly in **January**, indicating seasonal demand
- Majority of transactions are **B2B and Cleared**
- Sales performance is **concentrated among a few top salesmen**
- There is a **large gap between top and low performers**

---

## Challenges Faced
- Missing and null values in key columns
- Inconsistent data formats and special characters
- Complex sales values (e.g., combined amounts)
- Multiple status categories requiring filtering
- Data cleaning across multiple tools (Excel & Python)

---

## Conclusion
The business is **highly dependent on visa services** for revenue generation.  
Ticketing contributes additional support but has growth potential.

---

## Recommendations

- Increase ticket sales through **bundled travel packages (ticket + visa)**
- Learn from **top-performing salesmen** and share best practices
- Improve the performance of low-performing staff through training
- Strengthen **B2B partnerships**
- Implement **performance-based incentives**
- Use data for **seasonal planning and demand forecasting**
